# ⛏️ 博斯科终端 (非官方 DRG 任务追踪器)

[ [English](./README.md) | [한국어](./README.ko.md) | **中文** ]

## 🛠️ Tech Stack

* **前端**: Flutter (Dart)
* **数据管道**: Python (数据抓取) + GitHub Actions (自动化)
* **存储**: Shared Preferences (本地设置)
* **架构**: 模块化 UI 组件
